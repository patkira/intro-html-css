What Are HTML & CSS?

HTML - HyperText Markup Language

Use for structuring content such as headings, paragraphs, images

CSS - Cascading Style Sheets

Use for styling content such as font colour, positioning content

HTML elements

  • Heading - <h1>, <h2>
  • Text - <p>, <strong>
  • Link - <a href="http://www.oswd.org/">
  • Image - <img src="mdn-logo-sm.png" alt="">
  • List - <ul>, <ol> with <li>

HTML elements

  • Layout - <header>, <footer>
  • Content layout - <aside>, <nav>
  • Container - <div>, <span>

Styling our page

CSS - Cascading Style Sheets

It is used for styling html page and it should be separated from the HTML.

Keyword: Cascading

A simple CSS rule


// Selecting element
p { ... }

// CSS properties
p {
  color: ...;
  font-weight: ...;
  text-align: ...;
}

// Assigning properties some values
p {
  color: red;
  font-weight: bold;
  text-align: center;
}

CSS selectors


/* Tag */
h1 { ... }

/* Class name */
.example { ... }

/* ID */
#example { ... }

/* All paragraphs with class name of "example" */
p.example { ... }

/* All elements inside a paragraph with class name of "example" */
p .example { ... }

Display inline, inline-block & block

Inline - sits next to other elements, ignores width or height.

Inline-block - sits next to other elements, accepts width or height.

Block - breaks past other elements, takes as much width.

CSS specificity

  1. Inline styles - <h1 style="color: #fff;">
  2. IDs - #example
  3. Classes and attributes - .example
  4. Elements - h1, span, ul

What happens if you have 2 contradictory rules that target the same elements?


p {
  color: red;
}

p {
  color: blue;
}

Box sizing

Content box - width and height = what is specified, padding and border are added on.

Border box - padding and border are included in specified width and height.
