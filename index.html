What Are HTML & CSS?

HTML - HyperText Markup Language

Use for structuring content such as headings, paragraphs, media such as images

CSS - Cascading Style Sheets

Use for styling content such as font colour, positioning content

HTML elements

  • Headings - <h1>, <h2>
  • Text - <p>, <strong>
  • Link - <a href="http://www.oswd.org/">
  • Image - <img src="mdn-logo-sm.png" alt="">
  • Text - <p>, <strong>
  • List - <ul>, <ol> with <li>

HTML elements

  • Layout - <header>, <footer>
  • Content layout - <aside>, <nav>
  • Container - <div>, <span>

Styling our page

CSS - Cascading style sheets

It is used for styling html page and it should be sparated from the HTML.

Keyword: Cascading

A simple css rule


// Selecting element
p { ... }

// CSS properties
p {
  color: ...;
  font-weight: ...;
  text-align: ...;
}

// Assign properties some values
p {
  color: red;
  font-weight: bold;
  text-align: center;
}

CSS selectors


// Tag
h1 { ... }

// Class name
.example { ... }

// Id
#example { ... }

// Select all paragraph with class name of "example"
p.example { ... }

// Select all element 'inside' paragraph
// with class name of "example"
p .example { ... }

Inline, inline-block, block

Inline - sits next to other elements, ignores width or height.

Inline-block - sits next to other elements, accepts width or height.

Block - breaks past other elements, takes as much width.

Content box - width and/or height = what is specified, padding and border are added on.

Border box - padding and border are included in specified width and/or height.
